# TAR-09

Practica de WiFi con servidor HTTP en NodeMCU ESP8266. Desde una pagina web servida por la placa se selecciona un logo y el ESP8266 lo dibuja en una pantalla OLED.

## Motivo de la tarea

El reporte `Tar09_Cripto` busca entender el control de hardware desde un navegador mediante un servidor HTTP embebido en una red local. Tambien compara esa arquitectura LAN con una solucion IoT en la nube usando Firebase Realtime Database y GitHub Pages para lograr control fuera de la red local.

Reporte: [Tar09_Cripto.pdf](../../homeworks/Tar09_Cripto.pdf)

## Proyecto

- `tar-09-wifi-oled-httpserver/`: firmware completo con servidor web, pagina HTML embebida y dibujos en OLED.

## Requisitos

- PlatformIO.
- NodeMCU ESP8266.
- Pantalla OLED SSD1306.
- Red WiFi local.
